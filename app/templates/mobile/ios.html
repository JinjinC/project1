如何安装 OneAPM ？

很开心听到这个问题！
以下是 iOS Agent 安装指南。

1. 创建 OneAPM 账号

2. 点击 iOS 按钮，然后为你的应用命名

A CHEESY NAME

不喜欢？没关系，你随时都可以修改

3. 下载 OneAPM iOS SDK

OneAPM_iOS_Agent_1.x.zip

注：OneAPM iOS Agent 目前支持 iOS 6.0 及以上版本；Agent 也能放心安装至 iOS 6.0 中，只是无法查阅该部分设备的数据信息。

4. 解压 OneAPM iOS SDK

解压 SDK，然后将 “OneAPMAgent.framework” 从 Finder 拖至要添加的 Xcode project 中（放在 Project Navigator 窗口的 Project 上）。

收到提示后，选择 “Copy items into destination” 和 “Create folder references...“。

5. 在 Linker Settings 中添加以下 3 个 Libraries

在项目导航窗口内点击你的 Project，并选中你的 App，然后选择 “Build Phases" 选项卡。

打开 "Link Binary with Libraries" 列表。点击添加：

CoreTelephony.framework
SystemConfiguration.framework
libz.dylib

6. 在 AppDelegate上 启动 Agent

在 AppDelegate.m 文件头部引入 OneAPM Agent

#import<OneAPM/OneAPM.h>

在 application:didFinishLaunchingWithOptions: 方法的第一行，添加如下代码

[OneAPM startWithApplicationToken:@"generate app token by creating an app name"];

7. 运行应用程序

清除 Project，并重新在模拟器或设备中启动应用程序，开始应用性能管理

在 application:didFinishLaunchingWithOptions: 方法的第一行，添加如下代码

[OneAPM startWithApplicationToken:@"generate app token by creating an app name"];
