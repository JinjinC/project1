Agent 计算

在不同语言中，Agent 计算方式不一样。一个 Agent 等于，在基数范围内的度量依据的个数。如：Java 中，一个 Agent 数量等于一个 JVM 数量。

语言 度量依据 基数
Java JVM 1
.NET Windows OS 1
PHP Web Server 1
Node.js Node.js Process 10
Python Python Process 10
Ruby Ruby VM 1

常见问题

如何收费

免费版永久免费。

专业版按照 Agent 进行收费；Agent 是指插入至您的应用程序中的代理。不同语言的 Agent 度量方式不一样，请查看 Agent 计算。

企业版的收费情况，请电话联系 OneAPM 销售人员：400 - 659 - 1230。

是否可以进行试用？

新注册用户，都享有专业版功能免费试用 15 天的服务。

当专业版试用过期后，会自动转为相关产品的免费版；而免费版永久免费。

如何停止服务？

卸载 OneAPM Agent，至 Agent 不再向 OneAPM 服务器发送数据，OneAPM 就会在下个月停止收费。

卸载时，如存在技术疑问，请邮件联系 OneAPM 支持人员：support@oneapm.com。

是否存在优惠政策？

OneAPM 针对年付用户，会做一系列的优惠。具体事宜，请电话联系 OneAPM 销售人员：400 - 659 - 1230。

OneAPM 也会在不同季度，提供相应的优惠政策，请持续关注 OneAPM 相关邮件。

OneAPM 针对创业团队和合作伙伴，也会提供相应的优惠政策。优惠政策的答疑，请电话联系 OneAPM 销售人员：400-622-3101。

专业版和企业版的差别在哪？

专业版和企业版在功能上的差别主要体现在：对传统类和大型企业的技术框架的支持，面向业务的性能监控，以及定制化的需求。

企业版的服务更好，支持 OneAPM 支持人员的到场服务：包括部署、特殊实施需求的满足、现场答疑等等。企业版还支持软硬件一体化的购买模式，私人的数据中心，确保您的性能数据更加安全。

企业版的购买和答疑，请电话联系 OneAPM 销售人员：400 - 659 - 1230。
