OneAPM for .NET

OneAPM for .NET 能夠深入到所有 .NET 應用內部完成應用性能管理和監控，包括代碼級別性能問題的可見性、性能瓶頸的快速識別與追溯、真實用戶體驗監控、伺服器監控和端到端的應用性能管理。

安装指南

端到端的性能可見性

按照瀏覽器和地理信息 2 種分類方式，對真實用戶的頁面載入時間等性能數據進行查看，快速區隔出需要優化性能的瀏覽器類型和區域用戶。

  • 比較不同版本的應用性能
  • 通過對吞吐量和集群的性能優化，來避免硬體成本消耗

支持主流框架/平台

OneAPM支持所有 .NET 語系： VB.NET、C# 和 C++/CLI for .NET 2.0 及以上版本。OneAPM 支持主要框架：MVC 2、MVC3、MVC4、ASPs 和 WebServices。

除開真實用戶體驗監控以外，OneAPM for .NET 可以達到代碼級別的性能可見性，如 SQL 語句的性能分析和問題定位。

OneAPM 可以監測私有數據中心中的應用，和部署於公共雲上的應用，包括：Microsoft Azure 和 Azure Cloud Services。

協同合作，只在 OneAPM

OneAPM 給團隊中每個角色分配不同許可權，管理應用性能，協同合作確保應用長期處於健康穩定狀態。通過設置報警觸發器和通知方案，確保在錯誤發生和性能出現問題時，能夠通知相關人員進行修復；健全的 Traces 記錄能夠幫助相關人員快速定位性能瓶頸。

1. 創建 OneAPM 賬號

2. 下載 OneAPM .NET Agent 安裝包

OneAPMAgent.msi

3. 完成安裝，並重啟 IIS

當授權提示出現時，輸入我們提供的授權碼：

tHisIsnoTailCenseKEYbyANymEAnsSiLLy

如果您沒有選擇立即重啟，您可以打開 CMD 進行手動重啟：

iisreset

4. 見證奇蹟的時刻！

幾分鐘後，你的應用就會開始向 OneAPM 發送數據。
